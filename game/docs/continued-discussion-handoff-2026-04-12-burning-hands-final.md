## 一、G型定义修正

**原定义（v0.3）**：`G | 扇形散射 | 多枚弹体从扇形区域同时发射`

**新定义（v0.8）**：`G | 扇形区域 | 以施法者为中心，向前方扇形区域扩散的火焰/能量区域；区域内目标同时受击`

核心差异：G型不是"多枚弹体"，而是**一整片连续的扇形火焰片区**，参考 D&D Burning Hands。

## 二、reg07 Burning Hands 实现

**文件**: `six-fighter-web/src/scenes/SkillDemoScene.ts` 行 631+

### 技能参数

| 参数 | 设计值 |
|------|--------|
| 扇形角度 | 90°（±45°） |
| 最大扩散半径 | 120 px |
| 扩散时长 | 350 ms |
| 持续时长 | 400 ms |
| 总时长 | 750 ms |

### 视觉规格

**底层**：大范围模糊红光晕，半径在 60%~160% 之间随机波动，边缘极度不规则

**中层**：32 个有机火焰粒子，沿扇形区域分布，每帧独立计算：
- 位置抖动（角度 ±0.25、距离 ±0.18）
- 尺寸呼吸（速度相位错开）
- 颜色渐变（靠边缘偏橙红，靠内偏深橙）
- 三层叠加：外层光晕 / 中层实色 / 内层亮核

**最前端**：12 个深橙色大斑块，沿前端随机分布，随时间抖动，模拟火焰前锋的不均匀突出感

**无任何 stroke 线条**，无任何整齐几何边界

### 命中检测

- 每约 80ms 判定一次扇形内所有目标
- 目标在扇形角度范围内 + 距离小于当前半径 × 1.1 即触发 hit feedback
- 同一目标只判定一次（Set 去重）

### 迭代历史

1. **第一版**：多枚独立子弹飞散 → 被用户否定，"G型不是子弹"
2. **第二版**：几何扇形 + 钢针状楔形火舌 → 用户否定，"边缘太整齐"、"火舌像钢针"
3. **第三版（当前）**：有机粒子群 + 无几何线条 → 用户认可
