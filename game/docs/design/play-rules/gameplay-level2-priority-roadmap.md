# Gameplay Level-2 Priority Roadmap

Status: Draft
Version: v0.1
Owner: Design
Last Updated: 2026-03-20
Scope: Confirms Level-2 vertical expansion order and entry criteria for phase 1 gameplay modes.
Related: docs/design/play-rules/gameplay-framework-phase1.md

## 1. Confirmed Priority Order (Risk First)
1. Ruins Escape L2
2. Castle Defense L2
3. Deep Dungeon L2
4. Mainline Chapter 1-5 L2

## 2. Why This Order
- Ruins Escape has the highest retention and fairness risk due to death-loss and extraction rules.
- Castle Defense has medium-high system coupling risk between wave pacing and structure economy.
- Deep Dungeon has scaling risk but lower immediate frustration risk.
- Mainline chapter deepening can safely proceed after independent-mode rule stability.

## 3. L2 Entry Criteria
- Level-1 framework doc exists and is internally complete.
- Matching values CSV placeholder exists with baseline fields.
- Cross-mode rules have no conflict with combat freeze terms.
- At least three open items are listed for each mode's L2 breakdown.

## 4. Expected L2 Deliverables per Mode
- Logic document with sub-loop decomposition.
- Mode-specific fail/reward algorithm definition.
- Values CSV expansion with tiered parameter groups.
- Validation checklist for readability, fairness, and progression impact.
