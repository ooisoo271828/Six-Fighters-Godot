- 已从“顶视角思路”改为“侧视角思路”并重制多轮。

  - 不规则外形；
  - 上粗下细；
  - 外围空气碎片粒子；
  - 尾迹贴地。
- 该技能 **仍在可视微调阶段**（明日优先继续）。
